Applied Mathematics for Intelligent Systems is a curated research and implementation repository dedicated to the mathematical pillars of modern Artificial Intelligence. In an era where "black-box" models are common, this project focuses on White-Box Understanding—mastering the underlying physics of information that powers everything from Large Language Models (LLMs) to Human-Centric Cognitive Architectures.This repository is not just a collection of notes; it is a computational lab where abstract mathematical concepts are translated into optimized Python code. It is specifically structured to align with the interdisciplinary demands of Cognitive Science, Advanced Software Engineering, and Digital Humanities.🎯 Key ObjectivesDeconstruct Complexity: Breaking down the "math-heavy" barriers of Deep Learning into first-principle logic.Bridge Theory & Code: Moving from LaTeX-defined formulas ($\LaTeX$) to pure NumPy/Python implementations without relying on high-level abstractions.The Bayesian Perspective: A deep dive into the "Bayesian Brain" hypothesis—applying probability theory to mimic human decision-making and perception.Academic Validation: Serving as a technical portfolio for elite European M.Sc. programs, demonstrating proficiency in the Scientific Disciplinary Sectors (SSD) required for AI research.🧩 Core Mathematical DomainsLinear Algebra: Mapping feature spaces, understanding high-dimensional transformations, and Singular Value Decomposition (SVD).Multivariate Calculus: Mastering the "Learning" part of Machine Learning through Gradients, Jacobians, and loss function optimization.Bayesian Statistics: Shifting from Frequentist views to the logic of uncertainty, essential for AI Ethics and Cognitive Modeling.Information Theory: Quantifying "Surprise" and Entropy to build more efficient NLP architectures.
